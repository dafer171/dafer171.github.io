Site Purpose

This website has the following objectives:

  • Establish relationships between all providers of tourist activities.

  • Collect events and activities offered by tourism activity providers.

  • Create a place where all those activities and events can be displayed.

  • Organize information and display it in a way that makes it easy to attend activities that people find interesting.

  • Allow citizens to report on upcoming events
